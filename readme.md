# Student Management System (C++)

##  Description
A console-based Student Management System built using C++.  
It manages student records using file handling and menu-driven operations.

##  Features
- Add Student
- Update Student
- Delete Student
- Display All Students
- File Handling (Persistent Storage)
- Menu-driven Interface

## Technologies Used
- C++
- File Handling
- OOP Concepts

##  How to Run
1. Compile the code:
   g++ main.cpp -o sms
2. Run:
   ./sms

##  File Storage
- Data is stored in `students.txt`
